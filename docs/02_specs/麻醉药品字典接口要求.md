# 麻醉药品字典接口要求

> 接口前缀：`/api-samis/pc/v1/anesthesiaDict/*`（与 [麻醉记录单后端接口要求](./麻醉记录单后端接口要求.md) 一致）

## 原则

| 规则 | 说明 |
|---|---|
| 字典推荐 | `defaultIsSpecial` 仅用于弹窗**默认勾选推荐** |
| 医生确认 | 保存用药时以请求体 `isSpecial` 为准 |
| 禁止强制 | 后端不得仅凭 `drugName` 覆盖前端显式传入的 `isSpecial` |
| 缺省补齐 | 仅当请求未传 `isSpecial` 时，可按字典默认补齐 |

## 1. 获取麻醉药品字典

```http
GET /api-samis/pc/v1/anesthesiaDict/getDrugDict
```

查询参数（可选）：

| 参数 | 说明 |
|---|---|
| keyword | 名称/编码模糊 |
| drug_category | 药理分类 |
| special_category | 特殊用药分类 |
| default_is_special | 是否默认推荐特殊 |
| enabled | 是否启用 |
| page / page_size | 分页 |

响应 `data[]` 字段（camelCase）：

```json
{
  "drugId": "drug_001",
  "drugCode": "NE",
  "drugName": "去甲肾上腺素",
  "drugAlias": "NE",
  "drugCategory": "vasoactive",
  "defaultMode": "continuous",
  "defaultRoute": "泵注",
  "defaultDoseUnit": "mg",
  "defaultRateUnit": "μg/kg/min",
  "defaultIsSpecial": true,
  "specialCategory": "vasoactive",
  "specialReasonTemplate": "维持血压",
  "specialDisplayTemplate": "{time} {drugName} {dose}{doseUnit} {route}，{reason}",
  "allowManualOverride": true,
  "isHighAlert": true,
  "isRescueDrug": false,
  "isVasoactive": true,
  "isAnticoagulant": false,
  "isObstetricDrug": false,
  "isElectrolyteDrug": false,
  "enabled": true,
  "sortOrder": 10,
  "remark": ""
}
```

前端 mock：`backendFetch` 已用 `seedDrugDict` + `drugDictItemToApi` 返回上述结构。

## 2. 新增/编辑麻醉药品字典

```http
POST /api-samis/pc/v1/anesthesiaDict/saveDrugDict
```

- 支持新增与编辑（body 含 `drugId` 为编辑）
- `defaultIsSpecial` 为推荐值，非强制
- `allowManualOverride` 默认 `true`
- 需写操作日志（后端待实现）

## 3. 停用药品字典

```http
POST /api-samis/pc/v1/anesthesiaDict/disableDrugDict
```

- 逻辑停用：`enabled = false`
- 历史麻醉记录保留 `drug_name` 快照，不受停用影响

## 4. 获取特殊用药分类

```http
GET /api-samis/pc/v1/anesthesiaDict/getSpecialDrugCategories
```

```json
[
  { "value": "vasoactive", "label": "血管活性药" },
  { "value": "rescue", "label": "抢救药" },
  { "value": "hemodynamic", "label": "血压/心率处理" },
  { "value": "anticoagulant", "label": "抗凝/抗凝拮抗" },
  { "value": "obstetric", "label": "产科特殊用药" },
  { "value": "electrolyte", "label": "电解质/内环境处理" },
  { "value": "allergy", "label": "抗过敏/激素处理" },
  { "value": "other", "label": "其他" }
]
```

## 5. 获取用药默认推荐（可选，后期扩展）

```http
GET /api-samis/pc/v1/anesthesiaDict/getDrugRecommend?drug_id=&drug_name=&anesthesia_method=&surgery_type=&patient_age=
```

响应示例：

```json
{
  "drugId": "drug_001",
  "drugName": "去甲肾上腺素",
  "recommendMode": "continuous",
  "recommendIsSpecial": true,
  "specialCategory": "vasoactive",
  "reasonTemplate": "维持血压",
  "allowManualOverride": true
}
```

当前阶段：前端可直接用 `getDrugDict` 默认字段完成推荐；本接口 mock 已实现。

选药后弹窗在「药品选择」区以 chip 展示系统推荐（普通用药 / 特殊用药），医生可在「特殊用药记录」区覆盖，见 `MedicationLineForm.vue`。

## 与用药保存的关系

用药保存见 [麻醉记录单后端接口要求](./麻醉记录单后端接口要求.md) 中 `saveMedication` / `batchSaveMedications`：`isSpecial`、`specialCategory`、`specialReason` 以医生最终提交为准。
