# 麻醉记录单用药显示规则

> 与 `src/services/medicationDisplayRules.ts` 实现一致；**是否画线**与 **是否特殊用药说明** 相互独立。

## 1. 是否划线

```text
持续发生的 → 时间轴画线段
瞬间发生的 → 仅时间点标记
```

判断（代码 `shouldRenderAsLine`）：

| 条件 | 画线 |
|---|---|
| `mode = 持续泵入`（continuous） | 是 |
| 有 `start_time`/`time` 与 `end_time`/`stop_time`，且结束晚于开始 | 是 |
| `mode = 单次用药`（single） | 否 |
| `mode = 间断追加`（intermittent） | 否（多个时间点，不连成一线） |

**禁止**用 `is_special` / `highAlert` 决定是否画线。

## 2. 是否进入特殊用药说明区

```text
is_special = true → 写入纸面「辅助及特殊用药」编号说明
is_special = false → 不写入该区
```

与 `mode`、是否画线无关。`highAlert` 仅表示高警示药品核对，不等同于特殊用药说明。

### 禁止逻辑

```text
❌ drug_name 命中特殊药 → 强制写入特殊用药区
❌ mode = intermittent → 自动 is_special
❌ drug_category 特殊 → 不允许医生取消
❌ highAlert → 等同特殊用药编号
```

### 推荐与确认流程

```text
药品字典 default_is_special / special_reason_template
  ↓
选药后弹窗自动推荐勾选（可取消）
  ↓
医生最终确认 is_special
  ↓
保存用药记录 & 渲染
```

实现：`src/services/drugDictRecommend.ts`（`buildMedicationLineRecommendPatch`）、`LiveAnesthesiaSheet` 中 `isSpecialUserTouched`。

## 3. mode 与 is_special 组合

| mode | is_special | 用药时间轴 | 特殊用药区 |
|---|---|---|---|
| continuous | false | 线段 | 无 |
| continuous | true | 线段 | 编号说明（双重展示） |
| single | false | 时间点剂量 | 无 |
| single | true | 时间点圈号/标记 | 详细说明 |
| intermittent | false | 多个时间点剂量 | 无 |
| intermittent | true | 多个时间点 + 可进说明区 | 可选说明 |

## 4. 字段约定

### 前端 `MedicationRecord`

| 字段 | 说明 |
|---|---|
| `mode` | `单次用药` / `持续泵入` / `间断追加`（持久化映射 single/continuous/intermittent） |
| `isSpecial` | 是否进入特殊用药区 |
| `specialNo` | 说明区编号 1–10，展示为 ①②③… |
| `time` / `eventTime` | 单次、间断事件时间 |
| `startTime` / `endTime` / `stopTime` | 持续区间 |
| `displayText` | 时间轴点标记自定义文案 |
| `specialCategory` | 特殊用药分类（vasoactive/rescue/…） |
| `specialReason` | 辅助及特殊用药区说明（医生可改） |
| `reason` | 兼容字段，与 specialReason 同步 |

### 字典默认值（可录入覆盖）

| 药品示例 | default_mode | default_is_special |
|---|---|---|
| 瑞芬太尼 | 持续泵入 | false |
| 丙泊酚 | 单次用药 | false |
| 舒芬太尼 | 间断追加 | false |
| 去甲肾上腺素 | 持续泵入 | true |
| 麻黄碱 / 阿托品 / 肾上腺素 | 单次用药 | true |
| 七氟烷 | 持续泵入 | false |

## 5. 液体与输血

晶体液、胶体液、血液制品、自体血回输仍按 `start_time`–`end_time` 在对应 band **画线段**，参与入量统计；与 `medication` 表无关，不受 `is_special` 影响。

## 6. 示例

| 录入 | 用药区 | 特殊用药区 |
|---|---|---|
| 瑞芬太尼 09:00–10:30 持续泵注 | 09:00–10:30 线段 | 无 |
| 去甲肾上腺素 09:30–10:40 持续，`is_special` | 线段 | ② 09:30-10:40 … |
| 罗库溴铵 09:05 50mg 静推 | 09:05 剂量点 | 无 |
| 麻黄碱 10:15 6mg，`is_special` | 10:15 ① | ① 10:15 麻黄碱 … |
| 舒芬太尼 09:10/09:40/10:15 间断 | 三个时间点 | 无 |
| 复方氯化钠 09:00–09:45 | 输液 band 线段 | 无 |
| 红细胞 10:00–10:40 | 输血 band 线段 | 无 |

## 7. 用药记录弹窗（临床录入）

弹窗标题为「用药记录」，宽度约 600px，分四区：

1. 药品选择（名称、类别、系统推荐 chip）
2. 给药方式（单次给药 / 追加一次 / 持续给药 + 一行模式说明）
3. 给药明细（单次/追加：时间+剂量+单位一行；持续：开始/结束+剂量）
4. 特殊用药记录（勾选后展开分类、说明、显示预览）

底部灰色规则条：`单次/追加 = 时间点，持续 = 自动线段，特殊用药 = 下方编号说明。`

实现：`MedicationLineForm.vue`、`RecordModalShell` size=`clinical`。
