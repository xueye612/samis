# 状态机字典

> 完整章节：[`../../术护手术麻醉管理系统完整功能规划表.md`](../../术护手术麻醉管理系统完整功能规划表.md) §21-23。

## 1. 手术主状态 (`surg_operation.status`)

```
pending_accept   待接收
   ↓ accept
accepted         已接收
   ↓ schedule
scheduled        已排台
   ↓ ready
waiting_enter    待入室
   ↓ enter_room
entered          已入室
   ↓ start_anes
anes_started     麻醉开始
   ↓ start_surgery
surgery_started  手术开始
   ↓ end_surgery
surgery_ended    手术结束
   ↓ end_anes
anes_ended       麻醉结束
   ↓ leave_room
left_room        出手术室
   ↓ enter_pacu
in_pacu          入 PACU
   ↓ leave_pacu
left_pacu        出 PACU
   ↓ complete
completed        已完成
```

特殊状态：

| 状态 | 含义 |
|---|---|
| `rejected` | 已退回（接收前）|
| `cancelled` | 已取消（入室前）|
| `aborted` | 非计划中止 |
| `cancelled_after_enter` | 入室后取消 |
| `cancelled_after_anes` | 麻醉开始后取消 |

## 2. 麻醉文书状态 (`anes_*.status` / `mr_document.status`)

```
none → draft → submitted → pending_sign → signed → pdf_generated → archived
```

| 状态 | 含义 |
|---|---|
| `none` | 未创建 |
| `draft` | 草稿 |
| `submitted` | 已提交 |
| `pending_sign` | 待签名 |
| `signed` | 已签名 |
| `pdf_generated` | 已生成 PDF |
| `archived` | 已归档 |
| `rolled_back` | 已回退（管理员操作）|

## 3. PACU 状态 (`anes_pacu_record.status`)

```
waiting_in → recovering → reach_discharge → discharged → completed
```

## 4. 计费状态 (`bill_charge.status`)

```
draft → submitted → his_success
                 ↘ his_failed → resubmitted
```

## 5. 文书签名状态 (`mr_document_sign.status`)

```
pending → signed
        ↘ rejected
```

## 状态变更入口

所有状态变更必须通过 Service 层 `transition($entity, $event, $context)` 集中实现：

- 校验当前状态 → 目标状态的合法性
- 写状态变更前后值 + 操作人 + 时间到 `sys_audit_log`
- 触发对应事件（如手术结束 → 自动生成 PACU 待入室任务）

禁止在 Controller / Model 中分散调用 `update(['status' => ...])`。
