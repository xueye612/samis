# 主文档集（精简版）

用途：只保留后续开发和事实核对最常用、最稳定的文档。

收口原则：
- 保留：主事实、功能总览、复用边界、前后端现状、数据库主文档、少量权威参考。
- 省略：历史计划、归档文档、重复汇总稿、早期原型说明。

## 目录说明

- `00_入口/`
  - `INDEX.md`：唯一总入口
  - `主事实.md`：事实裁决优先级

- `01_总览/`
  - `功能总览.md`
  - `复用边界.md`

- `02_现状/`
  - `前端现状.md`
  - `后端合并说明.md`

- `03_数据库/`
  - `数据库主文档.md`

- `04_主源参考/`
  - `已完成功能.md`
  - `已实现接口.md`
  - `数据表约定.md`
  - `状态机.md`

## 暂不纳入本目录的文档类型

- `docs/01_plan/*`（阶段规划，已降级）
- `docs/90_archive/*`（历史归档）
- `docs/06_workspace_merged/*`（过程性汇总稿）
- 外部仓库中的历史说明文档（保留在原处，按需查阅）
