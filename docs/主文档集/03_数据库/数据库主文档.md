## 9. 字段命名规范（生成 SQL 时的基础）

- 主键统一：id（bigint unsigned, primary key, auto_increment）
- 引用键：`<table>_id`（如 record_id、pacu_id、visit_id）
- 时间：`*_at`（datetime），事件内时间字段用 `time`（datetime）
- 枚举：使用 char/varchar 存储 code，配合 sys_dict（不强制用 enum 类型）
- 金额 / 量：decimal(10,2) 起步
- JSON 字段：value_json / snapshot_json / payload / score_json
- 软删：is_deleted（tinyint）
- 乐观锁：version（int）
- 审计：created_by / updated_by（bigint 关联 sys_user.id）
