# 麻醉数据库开发主文档

## OceanBase 与前端本地映射

前端 IndexedDB 数据库名：`samis_anesthesia_local_v1`

### 主记录

| IndexedDB | OceanBase（规划） | 说明 |
|---|---|---|
| records.local_id | anes_record.local_id | 前端 UUID |
| records.server_id | anes_record.id | 后端主键 |
| records.operation_id | operation_info.id | 手术通知单 |
| records.case_payload | 多表聚合 | 前端快照 JSON |

### 子表

| IndexedDB | 后端表（规划） |
|---|---|
| timeline_events | anes_timeline_event |
| medications | anes_medication |

### anes_medication（用药）

| 字段 | 说明 |
|---|---|
| `mode` | `single` / `intermittent` / `continuous` |
| `is_special` | 是否写入特殊用药说明；**不等同于**不画线 |
| `special_no` | 特殊用药区序号 |
| `special_category` | 特殊用药分类（vasoactive/rescue/…） |
| `special_reason` | 辅助及特殊用药区说明文案 |
| `event_time` | 单次/间断给药时刻 |
| `start_time` / `end_time` | 持续泵注/输注区间（持续用药必填） |

药品字典表（规划 `anes_drug_dict`）须含：`default_is_special`、`special_category`、`special_reason_template`、`allow_manual_override`、`default_mode` 等，见 [麻醉药品字典接口要求.md](../02_specs/麻醉药品字典接口要求.md)。

**业务规则**：特殊用药仅控制说明区；持续用药必须在时间轴保留 `start_time`/`end_time` 线段。液体/输血见 `anes_fluid`、`anes_transfusion`，按时间段画线。规则见 [../02_specs/麻醉记录单用药显示规则.md](../02_specs/麻醉记录单用药显示规则.md)。

### 仅前端

| IndexedDB | 说明 |
|---|---|
| sync_queue | 上传队列，无直接后端表 |
| settings | 页码/偏好 |
| audit_logs | 本地审计；后端完成后可映射 anes_audit_log |

## local_id 与 server_id

- 创建时仅 local_id
- pushBatch 成功返回 server_id
- 前端永久保留 local_id 用于幂等与追溯

## sync_queue

不写入 OceanBase 业务表；后端仅需处理 pushBatch 幂等。

## 设备数据

- monitor_raw → 监护仪原始数据表（高频）
- ventilator_raw → 呼吸机原始数据表
- vital_signs → 记录单显示点（打印/趋势图）

## 审计

本地 audit_logs 记录保存/同步动作；与后端审计字段对齐后双向同步（待后端）。
